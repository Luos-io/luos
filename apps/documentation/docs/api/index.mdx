# Software integrations

Luos provides many tools to interface, control, or monitor products made with Luos engine.

|  Tools                                 |  Description                                                                                           |
| :-----------------------------------:  | :----------------------------------------------------------------------------------------------------: |
| [SDK python: Pyluos](/docs/tools/gate) | A translation Luos app service allowing you to easily connect your computer to your hardware product.  |
| [SDK typescript](/docs/tools/gate)     |                                                                                                        |
| [PLatformIO](/docs/tools/boot)         | Update node through Luos Network                                                                       |
| [Arduino IDE](/tools/inspector)        | A Luos service application allowing a computer to inspect a product.                                   |
| [Espressif IDE](/docs/tools/monitoring)| API that allows you to monitor a Luos network                                                          |
| [RTOS](/docs/tools/pyluos)             |                                                                                                        |
| [ROS 1&2](/docs/tools/pyluos)          |                                                                                                        |
